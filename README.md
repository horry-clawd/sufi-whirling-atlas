# Whirling Atlas

Interactive nightly build exploring how Sufi centers and ideas traveled from the 10th to 20th century.

## Features
- Tailwind + vanilla JS single-page app
- Leaflet map with real CARTO dark tiles
- Century scrubber (10th–20th) controlling map, story, and poetic quote
- Optional sonic pulse via WebAudio for a “wow” interaction
- Static JSON data block drives routes, centers, and quotes
- Mobile-first layout with reduced-motion support
